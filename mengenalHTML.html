Mengenal Bahasa HTML

HTML atau Hyper Text Markup Language merupakan sebuah bahasa markah untuk membuat halaman web. Padanan HTML dalam bahasa indonesia adalah Bahasa Markah Hiper Teks (BMHT).

Adapun struktur html dalam pembuatan halaman web dibentuk oleh elemen - elemen html, dimana setiap elemen html dibentuk oleh tag pembuka dan tag penutup yang punya nama tag dan fungsinya tersendiri

Untuk menuliskan kode html membutuhkan text editor misalnya notepad atau notepad++, selanjutnya file disimpan dengan akhiran .html misalnya latihan.hml. Kalau sudah disimpan, cobalah untuk mengetesnya dengan mengakses file latihan.html tersebut di web browser.
